Objetivo da Missão

Criar um programa em Delphi que calcule a média ponderada de três números com pesos diferentes.

Média Ponderada = (N₁ × P₁ + N₂ × P₂ + N₃ × P₃) / (P₁ + P₂ + P₃)

Requisitos da Atividade

  • Interface gráfica com campos para os 3 números
  • Campos para os 3 pesos correspondentes
  • Botão para realizar o cálculo
  • Exibição clara do resultado da média ponderada
Esta atividade valerá 2 pontos na avaliação.
